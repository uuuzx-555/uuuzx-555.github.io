# 我所了解的信息收集

# 1 确认范围

在拿到一个测试的目标后，首先能够确定的是测试范围。

# 2 使用爱企查，天眼查，企查查对域名所有者（一般是公司）进行查询

可收集：

  • 法人

  • 公司电话信息（网站管理者或公司固定电话）

  • 邮箱信息（可从邮箱后缀，猜测是否存在邮箱服务器等）

  • 地址

  • 官网和其他网站备案信息（有可能其他域名也在同一台服务器上）

  • 股东信息，主要成员（一般主要人员都会具有公司中的系统很大一部分权限）

  • 具体的公司经营情况，和业务范围（可能对业务逻辑有用，至少要知道对方是做什么的，在你看来这个系统的逻辑是什么样）

# 3Whois 信息查询

whois 是用来查询域名的 IP 以及所有者等信息的传输协议。whois 就是一个用来查询域名是否已经被注册，以及注册域名的详细信息的数据库（如域名所有人、域名注册商），通过 whois 来实现对域名信息的查询。

# 4 子域名信息查询

  • 子域名是指二级域名，是一级域名（顶级域名）的下一级。如：qq.com 是个顶级域名，那么 mail.qq.com 和 lol.qq.com 是 qq.com 的两个子域。

  • 企业可能有多个、几十个甚至很多的子域名应用，因为子域名数量多，企业在人员和防护的投入可能会没有主站及时。攻击者在主站域名找不到突破口时，就可以进行子域名的信息收集，然后通过子域名的漏洞进行迂回攻击。

  • 查找子域名，子域名可能跟主站在同一个服务器或者同一个 C 段网络中，可通过子域名探测的方式，收集目标的子域名信息，通过查询子域名的 IP 信息来辅助判断主站的真实 IP 信息。

常见顶级域名 类别
.com 用于工商金融企业
.net 用于互联网信息中心
.edu 用于教育机构
.gov 用于政府部门
  • 子域名枚举工具

Layer 子域名挖掘机用于网站子域名查询，有服务接口、暴力破解、同服挖掘三种模式。

  • 使用搜索引擎

例如 site:baidu.com site:qq.com

# 5 招聘信息收集

招聘网站的招聘信息和求职简历有大量的人员相关信息，招聘信息中涉及招聘公司招聘人员相关的电子邮箱、手机号等相关信息，求职简历中求职人员姓名、手机号、电子邮箱、工作经历等非常详尽的个人信息，如果招聘网站存在安全漏洞，求职人员的简历就有可能造成泄漏。
