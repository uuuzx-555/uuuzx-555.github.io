# Github 图床

PicGo 是一个自动上传图片到图床的工具

PicGo 官网

PicGo 使用说明

# PicGo 使用步骤

首先要有一个 Github 账号

新建一个仓库，记下你的仓库名字

生成一个 token 用于 Picgo 操作你的仓库：

访问 https://github.com/settings/tokens

点击 Generate new token

在 repo 上打勾，翻到底部点击 Generate token 绿色按钮生成 token.

** 注意：** 这个 token 生成后只会显示一次！你要把这个 token 复制一下存到其他地方以备以后要用。

# 配置 PicGo

** 注意：** 仓库名的格式是 用户名/仓库 。仓库名是是第二步创建仓库的名字，比如我创建了一个叫做 test 的仓库，在 PicGo 里我要设定的仓库名就是 test。一般我们选择 main 分支即可。然后记得点击确定以生效，然后可以点击设为 默认图床 来确保上传的图床是 GitHub
